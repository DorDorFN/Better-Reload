# Reload Backend

Content inside of this repository is derived from LawinServerV2 and Reload backends. To use any of the content, you MUST have it downloaded and launched. This repository is PRIVATE, meaning none of the information inside of it should be shared. You will be kicked off the Github if you do. 

## Features
 * Matchmaker + queue (50%)
 * Arena & Tournaments (100%)
 * Discord auth (0%)
 * Donation commands (full locker, etc) (not now)
 * Custom routes (vbucks on kill, etc) (70%)
